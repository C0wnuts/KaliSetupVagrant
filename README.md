# KaliSetupVagrant

Setup an up to date kali Linux via Vagrant.

## Included tools:
* Onetwopuch
* Testssl
* Joomlavs
* Sublist3r
* Unicorn
* Sudomy
* Crosslinked
* Droopscan
* Dirsearch
* Dymerge
* Many wordlists (IntruderPayloads,finaldics,fuzzdb)
* CrackMapExec
* Empire
* DeathStare
* Docker
* ApkTool
* Xalan
* Tilix
* Geany
* gdebi
* Many dependencies
